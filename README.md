# 📊 SMART ATTENDANCE System

A student attendance management system built with **PHP**, **HTML**, and **CSS**.

---

## ✨ Features I Built

- ✅ Add, edit, and delete student records
- ✅ Mark daily attendance (Present / Absent / Late)
- ✅ View attendance by date
- ✅ Generate monthly reports
- ✅ Admin login with secure sessions

---

## 🛠️ Technologies I Used

| Technology | What I Did |
|------------|------------|
| **PHP** | Login system, session management, form handling, page logic |
| **MySQL** | Basic data storage (simple setup) |
| **HTML/CSS** | Clean, responsive user interface |
| **XAMPP** | Local testing environment |

---

## 🔐 What I Learned / Implemented

- Password hashing (`password_hash()`)
- Session checks to protect pages
- Form validation
- Organizing PHP code across multiple files
